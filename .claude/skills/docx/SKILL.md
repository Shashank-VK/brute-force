# Word (DOCX)

Use this skill for professional document work where structure, formatting, and presentation matter.

## When to Use
- Drafting a new document or template.
- Reformatting an existing Word document.
- Preserving styles, headers, footers, tables, and pagination.

## Procedure
1. Identify the document type and its formatting rules.
2. Structure the content before polishing styling details.
3. Apply consistent styles for headings, lists, tables, and sections.
4. Check for layout breaks, spacing issues, and readability.
5. Deliver the document with any notable formatting decisions called out.
