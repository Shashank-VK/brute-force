# Frontend Design

Use this skill for UI work that needs a clear visual direction, strong hierarchy, and a finished product feel.

## When to Use
- Designing a new interface or landing page.
- Improving spacing, typography, color, motion, or visual hierarchy.
- Replacing generic AI-looking layouts with a deliberate design system.

## Procedure
1. Define the target audience and the visual tone.
2. Establish typography, color, spacing, and component rules before building.
3. Build responsive layouts that hold up on desktop and mobile.
4. Use motion and backgrounds intentionally, not as decoration.
5. Review the result against common AI-design failure modes.
