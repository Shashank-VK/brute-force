# RAG Architect

Use this skill to design or improve retrieval-augmented generation systems with strong retrieval quality and evaluation discipline.

## When to Use
- Planning a new RAG pipeline.
- Reviewing retrieval, chunking, or embedding choices.
- Diagnosing poor answer quality or hallucinations.

## Procedure
1. Define the corpus, user questions, and success criteria.
2. Choose the ingestion, chunking, and embedding strategy.
3. Design retrieval and reranking to match the query patterns.
4. Add evaluation steps for recall, precision, and answer quality.
5. Review failure modes and mitigation controls.
