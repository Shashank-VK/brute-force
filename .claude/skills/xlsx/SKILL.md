# Excel (XLSX)

Use this skill for spreadsheet work that needs structure, formulas, formatting, and reliable workbook output.

## When to Use
- Creating a new workbook or report.
- Editing formulas, tables, styles, or multiple sheets.
- Producing data exports that need spreadsheet-friendly structure.

## Procedure
1. Identify the workbook goal and the data model.
2. Decide which sheets, tables, formulas, and formatting rules are needed.
3. Preserve cell types, formulas, and layout while making changes.
4. Validate the workbook structure and any formula dependencies.
5. Summarize the output and any assumptions made.
