# Claude API

Use this skill to build or debug software that calls Claude across supported programming languages.

## When to Use
- Setting up a new Claude API integration.
- Implementing streaming, tool use, or message formatting.
- Comparing SDK patterns across languages.

## Procedure
1. Pick the language SDK and target model behavior.
2. Define the request shape, system instructions, and tool schema.
3. Implement the call path, retries, and streaming or tool handling.
4. Validate the integration with a small end-to-end example.
5. Document the assumptions, limits, and any required environment variables.
