# Deep Research

Use this skill when the answer depends on gathering and reconciling multiple external sources.

## When to Use
- Researching APIs, frameworks, or technical choices.
- Comparing approaches, vendors, or implementation strategies.
- Producing evidence-backed recommendations.

## Procedure
1. Define the question and the decision it needs to support.
2. Gather multiple sources and prefer primary documentation when possible.
3. Cross-check conflicting claims before summarizing.
4. Synthesize the findings into a concise recommendation.
5. Note any uncertainty or gaps in the evidence.
