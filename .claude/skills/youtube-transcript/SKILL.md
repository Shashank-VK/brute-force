# YouTube Transcript

Use this skill to extract useful information from a YouTube transcript instead of manually watching the whole video.

## When to Use
- Summarizing a video.
- Extracting quotes, chapters, or action items.
- Comparing claims made in a talk, tutorial, or interview.

## Procedure
1. Obtain the transcript or transcript source.
2. Clean and segment the content into readable sections.
3. Extract the requested facts, timestamps, or quotes.
4. Summarize the key points with any notable context.
5. Call out if the transcript is incomplete or unreliable.
